Términos de Uso y Política de Privacidad

1. Uso de la Plataforma

La plataforma permite analizar videos de YouTube extrayendo datos técnicos y sociales con fines informativos. Al utilizar esta herramienta, el usuario acepta las condiciones aquí descritas.

1.1. Condiciones de Uso

  • La plataforma se ofrece "tal cual" sin garantías de disponibilidad o precisión absoluta de los resultados.
  • No se permite el uso de la plataforma con fines comerciales sin autorización previa.
  • Los datos analizados no pueden ser utilizados para infringir las políticas de YouTube u otras normativas legales.

1.2. Responsabilidad del Usuario

  • El usuario es responsable del uso que haga de los datos obtenidos mediante la plataforma.
  • No se permite el uso de la herramienta para recopilar información de terceros de manera ilegal o con fines no éticos.

2. Privacidad y Protección de Datos

Nuestra plataforma no almacena información personal de manera permanente. Los datos analizados se procesan en tiempo real y se eliminan tras su uso.

2.1. Uso de APIs de Terceros

Utilizamos la API de YouTube para obtener información de los videos, y la API de OpenAI para procesar comentarios.

2.2. Seguridad de la Información

  • No compartimos datos con terceros.
  • Las claves API están almacenadas de forma segura para evitar accesos no autorizados.
  • Se aplican medidas de seguridad para proteger la plataforma.

3. Limitaciones Legales

No se permite la descarga o distribución de contenido de YouTube sin la debida autorización.

4. Protección de la API Key

Las claves API están protegidas mediante cifrado y medidas de seguridad para evitar accesos no autorizados.

Nos reservamos el derecho de actualizar este documento en cualquier momento.
